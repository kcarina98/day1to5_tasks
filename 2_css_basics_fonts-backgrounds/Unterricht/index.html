Unterschied zwischen inline und block elementen

Der Unterschied zwischen inline und block ist, dass block elemente sind IMMER 100% breit wie der display! Inline Elemente sind nur so breit wie das Element eben ist.

Beispiel für Blockelemente

  • article
  • section
  • div
  • footer
  • Überschriften h1-h6
  • ol und ul
  • li
  • p

Beispiele für inline elemente

  • a
  • button
  • img
  • input
  • label
  • textarea
  • span
